Available on crate feature desktop only.
Expand description

Aggregate multiple types implementing SpaceElement into a single enum type to be used with a Space.

§Example

use smithay::desktop::space::space_elements;

space_elements! {
    /// Name of the type
    pub MySpaceElements<'a, C>; // can be generic if necessary
    Window=Window, // variant name = type
    Custom=&'a C, // also supports references or usage of generic types
}
