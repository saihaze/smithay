Xwayland Window Manager module

Provides an X11Wm type, which will register itself as a window manager for a previously spawned Xwayland instances, allowing backwards-compatibility by seemlessly integrating X11 windows into a wayland compositor.

To use this functionality you must first spawn an XWayland instance to attach a X11Wm to.
